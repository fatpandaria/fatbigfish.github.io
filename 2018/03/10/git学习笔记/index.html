git学习笔记

发表于 2018-03-10 | 阅读次数:

在用hexo搭建完个人博客主页后，我发现对于其中使用到的Git技术，我还不甚了解，所以我又去找到廖雪峰老师

的Git教程开始学习，所以便创建了这个日志便于记录学习笔记。

安装步骤省去。

安装完成后，需要进行设置，在命令行输入：

git config –global user.name “username”

git config –global user.email “email@example”
还记得刚开始使用hexo时，便是因为没有设置上面两项，导致部署出错。

初始化一个Git仓库，使用git init命令。

添加文件到Git仓库，分两步：

第一步，使用命令git add ，注意，可反复多次使用，添加多个文件；

第二步，使用命令git commit，完成。git commit -m “修改提示语句”。

要掌握工作区的状态，使用git status命令。

如果git status告诉你有文件被修改过，用git diff filename可以查看修改内容。

版本回退
git log 查看修改日志。commit id是一个SHA1计算出来的用十六进制表示的数字。
要进行版本回退，Git必须知道当前版本是什么，Git中用HEAD表示当前版本，上一个版本是
HEAD^，所以我们用指令git reset –hard HEAD^。
总结：

HEAD指向的版本就是当前版本，可以使用命令git reset –hard commit_id
进行版本之间的穿梭。
git log可以查看提交历史，以便确定回退到那个版本。
要重返未来，用git reflog查看命令历史，以便确定回到未来哪个版本。
